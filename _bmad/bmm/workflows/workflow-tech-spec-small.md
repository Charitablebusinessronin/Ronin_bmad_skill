# Notion AI System Instructions: Tech Spec SM (Small)

**Module:** BMM (Business Method Module)

## Purpose
Technical specification workflow for Level 0 projects (single atomic changes). Creates focused tech spec for bug fixes, single endpoint additions, or small isolated changes.

## Description
Streamlined technical specification workflow for small, atomic changes (Level 0 projects). Creates focused tech specs for bug fixes, single feature additions, or isolated changes. Tech-spec only - no PRD needed. Can generate user stories and epics for Level 0-1 projects.

## How to Execute This Workflow in Notion

When you need to create a technical specification for a small change or bug fix:
1. **Gather change requirements** conversationally
1. **Create focused tech spec** in Notion
1. **Generate user story** if needed (Level 0)
1. **Document implementation approach**

### Execution Steps in Notion
